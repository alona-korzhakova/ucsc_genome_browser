Description

This track represents a single tissue or a cell type within IMPACT predictions.
The IMPACT method provides cell-type-specific regulatory element activity profiles across 707 unique combinations of transcription factor-cell type pairs.
The model predicts the epigenetic regulatory activity related to a particular transcription factor in a given cell type.
IMPACT scores are probabilistic, ranging from 0 to 1.
While values closer to 1 signify stronger activity and values closer to 0 represent negligible activity, depending on your question, specificity of signal in one particular track may be more informative than comparing magnitudes of comparable signals.
For detailed information about the project, please visit our page on using IMPACT tracks on the UCSC genome browser.

By default, only half of the cell types within the BLOOD tissue are displayed. You are, though, encouraged to enable all of them. For other tissues, once displayed, all cell types will be depicted.

Methods

We generated the data using TF ChIP-seq datasets (which determine the locations where a transcription factor (TF) is binding to the DNA) from the Accessions in the table listed above. All these TFs have a DNA binding motif.
We scanned the genome for matches to this motif and created a training set consisting of bound motifs and unbound motifs.
Then, we collected over 5,000 epigenetic datasets including histone marks (H3K4me1, H3K4me3, etc), chromatin accessibility (ATAC-seq), chromatin contacts (HiC), and sequence-based annotations from ENCODE, Roadmap, and NCBI.
Next, we trained IMPACT models to find which epigenetic features can distinguish between TF motifs that are bound, as evidenced by ChIP-seq, from those that are unbound, as described in Amariuta et al 2019 AJHG.
Finally, we applied these models at base pair resolution to predict the likelihood that there is regulatory activity at this position related to the TF and cell type in the training data.

Data Access

IMPACT prediction tracks are in bigWig file format to be compatible with the UCSC Genome Browser.
To download these bigWig files, please visit the Published Histories page on Galaxy and paste user_eq:impact_predictions into the search bar.
While bigWig files are binary files, we have also released IMPACT track predictions in bedGraph file format for easier data manipulation and analysis.
Both file formats are publicly accessible for download on Galaxy.
